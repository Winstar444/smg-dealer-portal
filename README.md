# SMG DEALERS PORTAL

## About
SMG Dealer Portal is a role-based web application designed to manage customer,
dealer, and admin workflows through separate dashboards.

HOW to run this project in vs code:-

BEFORE ADD ENV FILE IN THE PROJECT FOR FUNCTIONING OF BACKEND :- 

1. Run  "npm install" in terminal
2. Run "npm run dev"  in the same  terminal .
3. OPEN another terminal run "cd backend".
4. Run "npm run dev"
